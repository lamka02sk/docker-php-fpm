## Pull
```
docker pull eworkssk/php-fpm:8.1
```

## Usage
Mount your web application anywhere and expose ports:
```
docker run -v /my-app:/my-app -p 9000:9000 -p 9001:9001 --rm eworkssk/php-fpm
```

## Exposed ports
- `9000`: PHP-FPM server
- `9001`: PHP-FPM server status
